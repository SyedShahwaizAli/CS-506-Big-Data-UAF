# CS-506 Big Data - University of Agriculture Faisalabad

Welcome to the official repository for **CS-506 Big Data** class tasks, assignments, and projects at the **University of Agriculture Faisalabad**.  

This repository contains organized tasks for each class session with codes, reports, and resources.

## Repository Structure

- `Task-01/` - First class task files and documentation
- `Task-02/` - Second class task files and documentation
- `Task-03/` - Third class task files and documentation
- `Resources/` - Lecture notes, datasets, and references

## How to Use

1. Navigate to the folder of the task you want.
2. Open the `README.md` inside each task folder for task-specific instructions.
3. Access the code or documents as required.
